
ADVANCED TO-DO LIST
-This is a simple and clean task management app I built to learn more about flask and JS. It started as a way to organize my own school and programming schedule, but I have since turned it into a full web app that anyone can use to stay organized.

What it does
-The app is split into two parts. First, there is a login and registration system that keeps your "Realm" private. Once you are in, you get a dashboard where you can add tasks, set dates for them, and mark them as finished.
-I designed it to be very fast and easy to navigate. It uses specific filters like "Today" and "Upcoming" so you can see exactly what needs to be done right now versus what can wait for later.

Features
-Private Access: You create your own username and password to get into your list.
-Sticky Notes: Your tasks appear as cards on a grid, making it easy to read everything at a glance.
-Smart Filtering: You can quickly view only your completed tasks or see what is due today.
-Live Hosting: The app is hosted online, so it works from any browser.

How to use it
-You can try out the app immediately without needing to download anything.
-Live Demo: Click the demo button on the main page or go directly to https://psychoshinigami.github.io/Adv-To-Do-List/index.html.
-Register: Create a new account if it is your first time.
-Manage Tasks: Start adding your goals and marking them off as you complete them.

Tech stack
-I made the frontend by using HTML, CSS, JS.
-And the backend using Flask by seperating them with routes.
